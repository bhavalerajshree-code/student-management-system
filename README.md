# Student Management System

A complete web-based Student Management System developed using PHP and MySQL for managing students, attendance, examinations, accounts, notices, and reports efficiently.

---

# 🚀 Features

- Student Registration & Management
- Attendance Tracking System
- Exam & Result Management
- Public Result Portal
- ID Card Generation
- Account & Fee Management
- Notice Management
- Admin Dashboard
- Export Reports
- Login Authentication
- Chat Module
- Responsive User Interface

---

# ⚙️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/your-username/student-management-system.git
```

2. Move project folder to `htdocs`

3. Start Apache & MySQL using XAMPP

4. Import the database file into phpMyAdmin

5. Run the project:

```bash
http://localhost/student-management-system
```
